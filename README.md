# chat-vue-project

A chat app for generating links to chat rooms.
Drop links, chat anonymously.

## Overview

Language/Tools:
* VueJS (Client)
* NodeJS (Server)
* SQLite (Database)
* Socket.io (Websocket)
* Docker (Deployment)
